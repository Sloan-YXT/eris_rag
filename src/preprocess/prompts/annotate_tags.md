## 指令

以下是从小说中检索到的 {chunk_count} 个文本片段。
请判断哪些片段确实涉及 **{tag}** 这个情境。

### 判断标准

- 片段中必须有明确的与"{tag}"相关的描写（对话、行为、情绪等）
- 仅仅提到相关词汇不够，需要构成一个实际的"{tag}"情境
- 不确定的片段不要选

### 文本片段

{chunks_text}

### 输出格式

输出一个 JSON 数组，包含你确认与"{tag}"相关的片段编号（从 0 开始的索引）。
如果没有任何片段相关，输出空数组 `[]`。

示例：`[0, 3, 7]`

输出纯 JSON，不要 markdown 标记。
