## 指令

以下是艾莉丝的全部行为规则，从原文提炼而来。
从中提炼出她的**核心身份模块**。

### 要求

生成 6 个模块，每个模块一个 JSON 对象，放在数组中。
每个模块的 prompt_text 用**因果链**表述——不是"她很暴躁"，而是"因为X经历，所以她在Y情境下会Z"。

### 模块定义

1. **core** — 本质内核（任何对话都需要）。最根本的 3-4 个驱动力和矛盾。控制在 250 tokens 以内。activate_on: ["always"]
2. **vulnerability** — 脆弱面。恐惧、不安全感、如何流露脆弱。200-400 tokens。
3. **combat** — 战斗与力量。对强弱、战斗、保护的态度。200-400 tokens。
4. **relationship** — 关系模式。面对不同类型关系时的行为差异。200-400 tokens。
5. **pride** — 自尊与好胜。贵族意识、竞争心、被评价时的反应。200-400 tokens。
6. **growth** — 成长轨迹。不同时期的状态差异、变化的原因。200-400 tokens。

### 输出格式

```json
[
  {
    "id": "core",
    "domain": "core",
    "activate_on": ["always"],
    "prompt_text": "你是艾莉丝·格雷拉特。..."
  }
]
```

输出纯 JSON，不要包含 markdown 代码块标记。
