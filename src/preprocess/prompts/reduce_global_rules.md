## 指令

以下是从《无职转生》中检索到的与 **{domain}**（{domain_description}）相关的艾莉丝原文片段。

请从这些原文片段中归纳艾莉丝在此领域下的**行为规则**。

### 要求

1. 以成长回归后的艾莉丝为基准描述行为
2. 如果某个行为随时间变化了，在 growth_trace 中记录变化轨迹
3. 每条规则必须有原文依据（在 evidence 中标注来自哪卷）
4. 输出 3-6 条规则，每条聚焦一个具体情境

### 输出格式

JSON 数组，每个元素：

```json
{
  "id": "beh_{domain}_001",
  "situation_tags": ["从预定义标签中选择"],
  "condition": "当[具体情境]时",
  "behavior": "她会[具体行为]，详细描述反应过程",
  "motivation": "因为[深层动机/过去经历导致]",
  "growth_trace": "少女期:... → 魔大陆:... → 回归后:...",
  "language_examples": ["示例台词1", "示例台词2"],
  "exclusions": ["她绝不会做的事"],
  "evidence": "来自第X卷的描写"
}
```

### 预定义标签

愤怒, 恐惧, 喜悦, 悲伤, 羞耻, 心动, 嫉妒, 孤独, 无力感, 焦虑,
逞强, 隐瞒, 保护, 被保护, 攻击, 逃避, 道歉, 感谢, 哭泣, 拒绝,
受伤, 战斗, 练剑, 学习, 日常闲聊, 吃饭, 被夸奖, 被侮辱, 被小看,
被关心, 被抛弃, 比试, 遇强敌, 竞争, 命令, 失去, 重逢, 分离,
对鲁迪乌斯, 对基列奴, 对家人, 对陌生人, 对敌人, 对弱者

输出纯 JSON，不要 markdown 代码块标记。
